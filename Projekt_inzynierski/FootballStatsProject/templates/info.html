Informations about application

Main information
Application is based on three tools: Elasticsearch, Logstash and Kibana. Combined, they can be very powerful in processing files and preparing visualisations.

Our system was based on data available on this site.
We are prepared to parse every csv file but here you have csv file on which main kibana dashboard is based.
Columns description and more informations about file is available in this file.
What can you do?
Prepare csv file with your statistics and upload it into application to get your charts done. If predefined dashboard isn't enough for you, you can make visualisations by your own on kibana site. If you find your work interesing, share it with others!
